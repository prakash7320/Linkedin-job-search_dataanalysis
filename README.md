# LinkedIn Job Market Analysis

## Overview
This project analyzes LinkedIn job postings to understand job market trends, in-demand skills, and city-wise hiring patterns using Python and Power BI.

## Objectives
- Collect LinkedIn job data
- Clean and parse skill tags
- Analyze skills by city and job role
- Build an interactive Power BI dashboard
- Provide insights for job seekers

## Tools Used
- Python (Pandas, BeautifulSoup, Matplotlib)
- Excel
- Power BI
- chat gpt

## Key Insights
- Top skills demanded across cities
- Skill vs Role analysis
- Top hiring companies
- Job demand trends

## Dashboard
An interactive Power BI dashboard with filters for:
- City
- Skills
- Job role
- Work type
